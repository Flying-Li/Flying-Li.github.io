二、String

结构：

String是Object的子类，它实现了Serializable、Comparable、CharSequence

| 实现了Serializable接口，那就意味着这个对象是可以串行化的，也就是说该对象 |

| 可以实现网络传输 | |实现了Comparable接口就可以比较String对象 |

  • String是一个final类
  • String最重要的一点是它的value，字符串有一个属性private finla char value[ ]也就是说，String本质上还是char
  • 注意private final char value[ ]的final指的不是数值的不可修改，而是value在指向一个地址之后，不可以再指向一个新的地址。其中地址空间内的单个数值是可以改变的
